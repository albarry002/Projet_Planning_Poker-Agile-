# Planning Poker - Application Agile

##  Présentation du projet

**Planning Poker** (ou *Scrum Poker*) est une application développée en **Python** permettant aux équipes agiles d’estimer la complexité de leurs tâches de manière collaborative.  
Chaque participant choisit une carte (valeur de la suite de Fibonacci : 1, 2, 3, 5, 8, 13...) pour estimer une tâche. Les votes sont ensuite révélés afin de favoriser la discussion et l’alignement de l’équipe.

---

## Langage et outils nécessaires


**Langage principal**

- Python → pour le back-end et la logique du Planning Poker.

**Framework web**

- Flask → léger, simple et idéal pour un projet étudiant.

    - Permet de créer un serveur web et des routes HTTP.

    - Gère la connexion entre le front-end et la logique Python.

**Front-end**

- HTML / CSS / JavaScript → pour créer les pages web et         l’interface utilisateur.

- Bootstrap (optionnel) → pour rendre l’interface plus jolie rapidement.

**Tests et documentation**

- pytest → pour tester le back-end et la logique métier.

- Doxygen → pour générer automatiquement la documentation depuis les commentaires Python.

**Environnement de travail**

- VS Code ou GitHub Codespaces → pour coder dans un environnement partagé.

- Git + GitHub → pour la gestion des versions et collaboration.
